# Texture Synthesis

## Install

### install rust

```bash
curl https://sh.rustup.rs -sSf | sh
```

and add

```bash
source $HOME/.cargo/env
```

to ~/.zshrc or ~/.bashrc

### prepare env

```bash
conda create -n texture python=3.8
conda activate texture
./setup.sh
```

## Run

```bash
python demo.py
```

## Enjoy it~
